チェックインのお題を考えてください。

1. 以下のコマンドを実行して過去の履歴とメンバー情報を取得:
```
python3 cli.py checkin
```

2. 以下の条件でお題を**5つ**提案してください:
- 過去に使ったものと被らない
- チームの心理的安全性を高める、軽くて答えやすいもの
- メンバーのひととなりが分かるもの
- 曜日に応じたトーン（週始め=軽め、週末=楽しめ等）
- 番号付きで表示

3. ユーザーに「番号で選んでください（どれも微妙なら `0` で再生成）」と伝える

4. ユーザーが `0` や「もう1回」「再生成」「やり直し」等と言ったら、別の5つを新たに提案する（さっき出したものとも被らないように）

5. ユーザーが番号を選んだら、以下で履歴に保存:
```
python3 cli.py checkin-save "選ばれたお題テキスト"
```
保存完了を伝えて終了。
