リファインメントの議題案を生成します。顧客・他ベンダーも参加する場なので、対外的な議題を中心に準備します。

1. 今週の議事録を読み込む:
```
python3 cli.py search-daily "" -d 7
```

2. 今スプリントのBacklogデータを取得:
```
python3 cli.py weekly
```

3. メンバープロフィール（`data/members/` 配下）を参照（特に顧客対応メモ）

4. 以下を出力:

### リファインメント議題案

以下の観点で議題を提案してください:

**スプリント振り返り系**
- 今スプリントの進捗状況で顧客に共有すべきこと
- Blocked/待ち状態で顧客側のアクションが必要なもの

**中長期スケジュール系**
- 今後のマイルストーンに影響しそうな課題
- スコープ調整が必要な事項
- 技術的な判断が必要で顧客と合意すべきこと

**次スプリント計画系**
- 次スプリントに入れるべき優先度の高い課題
- 見積もりや仕様確認が必要なバックログアイテム

**その他**
- 今週のチーム内共有やふりかえりで出た、顧客にエスカレーションすべき事項

### 各議題の補足
- 議題ごとに背景・論点・期待するアウトプットを簡潔に添える

5. 上記の出力内容を `data/minutes/YYYY-MM-DD/weekly-prep.md`（今日の日付）に保存してください。同じ日に再実行した場合は上書きしてください。
