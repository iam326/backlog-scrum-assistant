ふりかえりの議題案を生成します。

1. 今週の議事録を読み込む:
```
python3 cli.py search-daily "" -d 7
```
（今週の全議事録を取得）

2. 今スプリントのBacklogデータを取得:
```
python3 cli.py weekly
```

3. メンバープロフィール（`data/members/` 配下）を全員分読む

4. 以下を出力:

### 今週のふりかえり議題案

今週の議事録・Backlogの動き・メンバーの傾向から、ふりかえりで話すべきトピックを提案:
- Keep候補: 今週うまくいったこと
- Problem候補: 課題やモヤモヤしそうなこと
- Try候補: 改善提案

### メンバー別の気になるポイント
- 今週のコンディションや動きから、ファシリテーション時に気を配るべきこと
